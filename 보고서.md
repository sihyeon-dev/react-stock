# 심플 주식 대시보드 보고서

| 번호 | 항목 | 내용 |
|---:|---|---|
| 1 | 프로젝트명 | 심플 주식 대시보드 |
| 2 | 프로젝트 주제 | Alpha Vantage API를 활용해 미국 주식 티커를 검색하면 현재가, 등락률, 가격 정보, 1개월 종가 차트, 관련 뉴스를 보여주는 React 기반 주식 정보 조회 서비스입니다. |
| 3 | 기획 배경 / 목적 | 주식 정보 사이트나 증권 앱은 초보자가 보기에는 정보가 많고 복잡할 수 있습니다. 이 앱은 사용자가 관심 있는 미국 주식 티커를 입력하면 핵심 주식 정보만 간단히 확인할 수 있도록 만들었습니다. 또한 React 수업에서 배운 컴포넌트 분리, props, useState, useEffect, 조건부 렌더링, 배열 렌더링, API 요청과 에러 처리를 실제 프로젝트에 적용하는 것이 목적입니다. |
| 4 | 사용 API | 사용 API는 Alpha Vantage입니다. API 기본 주소는 `https://www.alphavantage.co/query`입니다. `GLOBAL_QUOTE`로 현재가, 등락폭, 등락률, 고가, 저가, 시가, 전일 종가를 가져옵니다. `TIME_SERIES_DAILY`로 최근 일별 종가 데이터를 가져와 1개월 차트를 만듭니다. `NEWS_SENTIMENT`로 검색한 티커와 관련된 뉴스, 출처, 요약, 감성 라벨을 가져옵니다. API Key는 `.env` 파일의 `VITE_ALPHA_VANTAGE_API_KEY`에 저장하고 코드에서는 `import.meta.env.VITE_ALPHA_VANTAGE_API_KEY`로 사용합니다. |
| 5 | 화면 구성 | 화면은 상단 제목 영역, 탭 메뉴, 종목 검색 화면, 관심종목 화면으로 구성했습니다. 종목 검색 화면에는 티커 입력창, 검색 버튼, 빠른 선택 버튼, 로딩/에러 메시지, 검색 결과 카드가 있습니다. 검색 결과 카드에는 현재가와 등락률, 고가/저가/시가/전일 종가, 1개월 종가 차트, 관련 뉴스 영역, 관심종목 토글 버튼이 있습니다. 관심종목 화면에는 저장한 티커 목록, 조회 버튼, 삭제 버튼이 있습니다. |
| 6 | 데이터 흐름 | 사용자가 티커를 입력하고 검색 버튼을 누르면 입력값의 공백을 제거하고 대문자로 변환합니다. 그 다음 `fetchQuote()`, `fetchMonthlyCandles()`, `fetchCompanyNews()`를 순서대로 요청합니다. 현재가 데이터는 `stockData`, 차트 데이터는 `chartData`, 뉴스 데이터는 `stockNews`에 저장합니다. 저장된 상태값을 기준으로 React가 결과 카드, SVG 차트, 뉴스 목록을 화면에 다시 렌더링합니다. 관심종목을 추가하거나 해제하면 `watchlist` 상태가 변경되고, `useEffect`를 통해 `localStorage`에 저장되어 새로고침 후에도 유지됩니다. |
| 7 | 컴포넌트 구조 | `App.jsx`는 전체 상태 관리, API 요청 실행, 탭 전환, 관심종목 추가/삭제를 담당합니다. `src/api/stockApi.js`는 Alpha Vantage API 요청 함수를 분리해 관리합니다. `StockSearch.jsx`는 검색 컴포넌트로 티커 입력창, 검색 버튼, 빠른 선택 버튼을 담당합니다. `StockInfo.jsx`는 종목정보 컴포넌트로 현재가, 가격 상세 정보, 1개월 SVG 차트, 관심종목 토글 버튼을 출력합니다. `StockNews.jsx`는 뉴스 컴포넌트로 관련 뉴스 목록을 출력합니다. `Watchlist.jsx`는 관심종목 컴포넌트로 관심종목 목록과 조회/삭제 기능을 담당합니다. `App.css`는 전체 레이아웃, 카드 스타일, 상승/하락 색상, 반응형 화면을 담당합니다. |
| 8 | 기대 결과 및 추가 예정 기능 | 완성 후 사용자는 미국 주식 티커를 검색해 현재 가격 정보와 최근 흐름, 관련 뉴스를 한 화면에서 확인할 수 있습니다. 관심종목을 저장해 자주 보는 종목을 다시 쉽게 조회할 수도 있습니다. 추가하고 싶은 기능은 종목명 자동완성, 더 다양한 기간의 차트 선택, 관심종목별 현재가 요약 표시, 다크 모드, 백엔드 서버를 통한 API Key 보호, 여러 종목 비교 기능입니다. |
| 9 | 참고 자료 | Alpha Vantage 공식 API 문서: https://www.alphavantage.co/documentation/  /  React 공식 문서: https://react.dev/  /  Vite 공식 문서: https://vite.dev/  /  프로젝트 파일: `src/App.jsx`, `src/api/stockApi.js`, `src/components/StockSearch.jsx`, `src/components/StockInfo.jsx`, `src/components/StockNews.jsx`, `src/components/Watchlist.jsx`, `src/styles/App.css` |
| 10 | 느낀점 | 이번 미니 프로젝트를 만들면서 API 데이터를 가져오는 과정이 단순히 `fetch`를 쓰는 것에서 끝나는 것이 아니라, 로딩 상태와 에러 상황을 함께 고려해야 한다는 점을 알게 되었습니다. 또한 데이터를 화면에 바로 출력하는 것보다 API 요청 함수, 검색 컴포넌트, 결과 카드, 관심종목 컴포넌트로 나누니 코드가 더 이해하기 쉬워졌습니다. 특히 `useState`로 화면 상태를 관리하고, `useEffect`로 관심종목을 저장하고 불러오는 과정을 통해 React의 상태 변화와 렌더링 흐름을 더 잘 이해할 수 있었습니다. |
