# learning-scope.md

## 1. 목적

이 문서는 웹프런트엔드실습 수업의 현재 학습 범위를 정리하기 위한 파일이다.  
Codex 또는 다른 AI 에이전트가 프로젝트를 도와줄 때, 이 문서를 기준으로 **현재 수업에서 배운 수준 안에서 구현 방향을 제안**하도록 한다.

참고 사이트: https://my-website-bsn.pages.dev/subject1/intro/

---

## 2. 과목 개요

이 수업은 웹 클라이언트 프로그래밍의 기본 개념을 익히고, React를 활용하여 기본적인 웹 프로젝트를 제작하는 능력을 기르는 것을 목표로 한다.

수업에서 다루는 큰 흐름은 다음과 같다.

- HTML, CSS, JavaScript 기반 웹 클라이언트 프로그래밍 복습
- React를 활용한 기본 웹 프로젝트 제작
- API 기반 통신과 간단한 Node.js 백엔드 개념 이해
- React Native 기반 크로스 플랫폼 앱 개발 구조 이해

---

## 3. 현재 학습 범위 요약

현재 사이트 기준으로 확인되는 학습 내용은 아래와 같다.

| 차시 | 주요 내용 | 프로젝트에서 활용 가능한 개념 |
|---|---|---|
| 1차시 | HTML / CSS / JavaScript 복습, React 개념, 개발환경 | 기본 웹 구조, 스타일링, JS 문법, React 시작 |
| 2차시 | 현대 JavaScript와 JSX | ES6 문법, JSX 작성 |
| 3차시 | React Element 이해와 JSX 연습 | JSX로 간단한 UI 표현 |
| 4차시 | 컴포넌트와 Props | 컴포넌트 분리, props 전달 |
| 5차시 | state와 useState | 상태 관리, 사용자 입력 처리 |
| 6차시 | 조건부 렌더링, 배열 렌더링, map, key | 로딩/에러/결과 화면 분기, 목록 출력 |
| 7차시 | 1~6차시 내용 정리 | 기본 개념 복습 및 문제 풀이 |
| 9차시 | 일정관리 앱 만들기 | 입력, 목록, 상태 변경을 활용한 미니 앱 구성 |
| 10차시 | React Hooks | useState, useEffect 등 기본 Hooks 활용 |
