### 구현 범위 기준

- React에서 `fetch`를 사용해 Finnhub Quote API를 직접 호출한다.
- API 키는 `.env` 파일에 `VITE_FINNHUB_API_KEY` 이름으로 저장한다.
- 프론트엔드에서 사용하는 API 키는 완전히 숨겨지는 것이 아니므로, 이 프로젝트는 학교 과제용 데모 프로젝트 기준으로 제작한다.
- 백엔드 서버, 데이터베이스, 로그인 기능, 복잡한 상태 관리 라이브러리는 사용하지 않는다.
- 주식 정보는 현재가, 등락폭, 등락률, 고가, 저가, 시가, 전일 종가 중심으로 표시한다.
- API 요청 실패, 잘못된 티커, API Key 누락 상황은 화면에 에러 메시지로 표시한다.
- 현재 학습 범위에 맞게 React, JavaScript, HTML, CSS, 기본 상태 관리, 기본 이벤트 처리 중심으로 구현한다.
